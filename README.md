# AI Website Copy Generator

## 📌 Business Chosen

Spin Salon, Bangalore

## 🎯 Objective

To create high-quality website copy for local businesses using structured prompts.

## 🧠 Prompt Strategy

This project uses structured prompts with:

* Business details
* Target audience
* Tone control

to generate clear and persuasive website content.

## ✍️ Output Includes

* Homepage copy
* Services section
* CTA sections

## 🛠 Tools Used

* ChatGPT

## ✨ Key Features

* Simple, human language
* Non-generic content
* Ready-to-use website copy

## 🚀 Outcome

This project demonstrates how AI can help local businesses improve clarity, trust, and customer conversions.
